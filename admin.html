How publishing works:
  • Save — caches edits in this browser only.
  • Publish to GitHub — commits data/profile.json directly to Khairul-Kabir/profile.kabir.io@main via the GitHub API. GitHub Pages redeploys in ~30s.
  • Download — downloads the JSON file (manual fallback).
On first publish you'll be asked for a fine-grained Personal Access Token (Contents: Read/Write on this repo). The token is stored in your browser's localStorage. Use the key icon in the topbar to forget it.
